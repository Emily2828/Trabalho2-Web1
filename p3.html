Informações sobre o Sistema

Objetivo do Sistema

Este sistema foi desenvolvido para auxiliar na gestão dos pontos de estacionamento de bicicletas no campus do IFCE Maracanaú, permitindo que estudantes e funcionários visualizem em tempo real a disponibilidade de vagas em cada bicicletário.

Como Funciona

Os pontos de estacionamento são classificados em três estados:

Disponível: Mais da metade das vagas livres

Quase cheio: Menos da metade das vagas livres

Lotado: Nenhuma vaga disponível

Acessibilidade

Este sistema foi desenvolvido seguindo princípios de acessibilidade web:

  • Contraste adequado entre texto e fundo
  • Navegação via teclado
  • Texto descritivo para todos os elementos
  • Estrutura lógica e hierarquia clara
